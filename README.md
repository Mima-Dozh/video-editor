
# video-editor  
### Возможности утилиты
№ | Описание
:---: | :---: 
 1 | Объединение видео   
 2 | Вырезание фрагментов видео 
 3 | Извлечие кадров из видео 
 4 | Создание клипа из картинок 
 5 | Изменение скорости видео

 7 | Эффект fade-out 
 8 | Сохранение изменений 
 9 | Отмена изменений
