- [x] Объединение видео

- [x] Извлечие кадров из видео 
- [x] Создание клипа из картинок 
- [x] Изменение скорости видео 
- [x] Эффект fade-in 
- [x] Эффект fade-out 
- [x] Поворот картинки
- [x] RGB матрица кадра видео
- [x] Сохранение изменений 
- [x] Отмена изменений 
- [x] Предпросмотр
- [x] Тесты
- [x] Проверка скорости звука и картинки
